1. Style your Week 2 HTML pages with CSS
2. Practice using the box model with padding, margin and borders
3. Experiment with colors, fonts, and typography

Assignment Details: Instruction from assignment portal
Part 1: Style Your Profile Page Take your profile html file from Week 2 and create a stylesheet to make it beautiful. File Names Your html file name (e.g profile.html) and style.css

Requirements • Link an external CSS file to your HTML • Use at least 3 different CSS selectors (element, class, ID) • Apply colors to text and backgrounds • Style typography (font-family, font-size, font-weight, line-height) • Use the box model: add padding, margin, and borders to elements • Style your images (add border-radius for rounded corners, when necessary) • Make links change color when hovered (use :hover pseudo-class)

Part 2: Style Your Recipe/Tutorial Page Create a separate stylesheet for your recipe html page.

File Names Your html files (e.g recipe.html) and recipe-style.css

Requirements • Link an external CSS file • Style your lists with custom colors and spacing • Use different background colors for different sections • Apply the box model to create clear visual separation • Center-align your headings using text-align • Make the page visually organized and easy to read

Part 3: Box Model Practice Create a simple HTML page demonstrating your understanding of the box model. File Names boxmodel.html and boxmodel.css Requirements • Create 3 boxes (div elements with class names) • Each box should have: different background color, padding, border, and margin • Use border-radius on at least one box • Add text inside each box explaining what it demonstrates
